PURPOSE

SYNOPSIS

function output=saturate(x, level)

DESCRIPTION 

saturate: Saturate between two level (bound limiter)
    Usage:output=saturate(x, level)
        output: output value
        x: input value
        level: level(1)==>lower bound, level(2)==>upper bound

    For example:
        x=linspace(0, 10);
        level=[3 5];
        y=saturate(x, level);
        plot(x, x, x, y, 'o-');
