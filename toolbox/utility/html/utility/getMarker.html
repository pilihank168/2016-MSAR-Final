PURPOSE

getMarker: Get marker from a rotating palette

SYNOPSIS

function marker=getMarker(index)

DESCRIPTION 

 getMarker: Get marker from a rotating palette
    Usage: marker=getMarker(index)

    For example:
        t=linspace(0, 2*pi, 21);
        for i=1:13
            line(t, sin(t)*i/13, 'marker', getMarker(i));
        end
