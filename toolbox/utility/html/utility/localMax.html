localMax

PURPOSE

LOCALMAX Find local maxima of a vector

SYNOPSIS

function index = localmax(x, plotOpt)

DESCRIPTION 

 LOCALMAX Find local maxima of a vector
    The output is a 0-1 index vector indicating where the local maxima are.
    Usage: index = localmax(x, plotOpt)

    For example:
        t = 0:0.2:2*pi;
        x = sin(t)+randn(size(t));
        index = localmax(x, 1);
