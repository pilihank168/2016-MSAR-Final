DESCRIPTION 

 barPlot: Plot of 1-D data as bars and texts
    Usage: barPlot(data, xTickLabel)

    For example:

        dataNum = length(data);
        xTickLabel={};
        for i=1:dataNum
            xTickLabel={xTickLabel{:}, sprintf('第 %d 筆資料', i)};
