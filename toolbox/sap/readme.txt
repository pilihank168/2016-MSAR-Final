This toolbox provides functions for SAP (Speech and Audio Processing)

You also needs to have the following toolboxes in order to use the SAP toolbox:
	Utility Toolbox
	Machine Learning Toolbox

All these toolboxes are available from Roger Jang's homepage at "http://mirlab.org/jang".
